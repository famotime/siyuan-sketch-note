# 闲笔 (Sketch Note)

思源笔记手写插件。在文档中嵌入一块手写画布，用触控笔或鼠标自由书写、绘图、标注，内容以矢量形式保存，随时可以再次编辑。

## 典型场景

**课堂 / 会议速记** — 手写比打字更快捕捉灵感。插入一个手写块，用画笔或荧光笔记录要点，支持多页分隔，结束后导出 PDF 存档。

**思路整理** — 用形状工具画流程图，用套索移动和重组内容，用文本框补充说明。9 种内置稿纸模板（方格、横线、康奈尔笔记等）帮助你快速排版。

## 功能一览

### 书写与绘图

| 工具 | 说明 |
|------|------|
| 画笔 | 基础手写工具，支持压感（需硬件支持） |
| 荧光笔 | 半透明标注，适合重点标记 |
| 橡皮擦 | 两种模式：像素擦除（精确）和整笔擦除（快速） |
| 形状 | 直线、箭头、矩形、椭圆、三角形 |
| 文本 | 在画布上添加可编辑文字 |

每个工具独立记忆粗细和透明度，切换回来时自动恢复上次设置。

### 编辑操作

- **撤销 / 重做** — 完整操作历史，支持快捷键
- **套索选择** — 自由套索或矩形框选，选区可移动、缩放、复制、删除、改色
- **清除** — 一键清空当前页

### 稿纸模板与背景

9 种内置模板：白纸、方格纸、横线纸、点阵纸、康奈尔笔记、会议记录、待办清单、周计划、月计划。

### 输入设置

- **仅触控笔** — 开启后手指触摸不会产生笔迹，防止误触
- **笔触压感** — 启用后笔迹粗细随书写力度变化（需硬件支持）
- **禅模式** — 隐藏工具栏，最大化书写区域
